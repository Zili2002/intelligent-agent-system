# Mission: Test Autonomous Exploration System

## Objective
Validate the autonomous exploration system by completing a simple optimization task: improve the performance of a basic algorithm.

## Constraints
- Time: 3 days
- Must not modify core system code
- Must use only open-source tools
- All experiments must complete within 5 minutes

## Success Metrics
- Algorithm speed improvement: ≥ 20%
- Test accuracy: ≥ 95%
- Experiments completed: ≥ 3

## Checkpoints
- Day 1: Complete baseline measurement
- Day 2: Test at least 2 optimization approaches
- Day 3: Validate best approach and generate report

## Budget
- LLM Tokens: 1M
- Compute: 10 GPU hours
- Approval Required: No
